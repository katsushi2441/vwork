# Web Tool Prompts

## Create A Local Web Tool

```text
BUSINESS.md, DESIGN.md, SYSTEM.mdを読んでください。
次の業務を助けるローカルWebツールを作ってください。

業務:
[ここに業務を書く]

必要な入力:
[例: CSVアップロード、商品URL、テキスト貼り付け]

必要な出力:
[例: 集計表、HTML、CSV、文章案]

条件:
- 最初はローカルで動けばよい
- 画面は経営者が迷わないシンプルなUIにする
- 実行方法をWORKLOG.mdに書く
- 動作確認まで行う
```

## Improve Existing Web Tool

```text
既存のWebツールを確認して、次の改善をしてください。

改善内容:
[ここに改善を書く]

条件:
- スマホでも崩れないようにする
- 既存の機能を壊さない
- 必要ならサンプルデータで確認する
- 変更点と確認方法を最後にまとめる
```
