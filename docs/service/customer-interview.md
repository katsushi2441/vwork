# Customer Interview

Use these questions before creating a customer VWork workspace.

## Business

1. What work takes too much time right now?
2. What mistakes happen repeatedly?
3. What work depends on one person?
4. What data is already available?
5. What would be useful within one week?

## Users

1. Who will run the workflow?
2. Who will read the output?
3. Who decides whether it is useful?

## Data

1. Is the data in CSV, Excel, database, website, API, PDF, or email?
2. Is there confidential data?
3. Can sample data be used for development?

## Output

1. Should the output be a file, page, dashboard, report, API, or automation?
2. Does it need to be published?
3. Does it need search, SNS, or analytics?

## First Implementation

1. What is the smallest useful result?
2. What can be skipped for the first version?
3. How will we know it worked?
