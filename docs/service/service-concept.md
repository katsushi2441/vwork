# VWork Service Concept

## One-Line Concept

VWork installs a vibe coding foundation on the customer's PC and delivers the
first working implementation for a real business issue.

## Japanese Positioning

完成システムを受託開発するのではなく、お客様自身がCodexと一緒に業務改善を進められる作業基盤を構築し、最初の課題解決コードまで作って引き渡す。

## What The Customer Buys

- a local AI-driven workbench
- structured business/design/system documents
- one working example for a real issue
- a way to continue improving with Codex
- optional companion support

## What The Customer Does Not Buy

- a fully completed custom system
- indefinite maintenance
- a black-box vendor dependency

## Seminar Connection

The seminar explains why AI should move from "thinking support" to "execution".
VWork is the practical foundation installed after the seminar.

```text
Seminar
  -> understand AI execution
  -> install VWork
  -> solve one real issue
  -> continue vibe coding internally
```

## Suggested Service Menu

### 1. Intro Seminar

For executives who need to understand AI-driven management and vibe coding.

### 2. VWork Setup Pack

Install the local environment and create the customer workspace.

### 3. First Issue Build Pack

Build the first useful implementation from a real business issue.

### 4. Companion Support

Monthly support for review, fixes, and next workflow design.

## Sales Message

Your company does not just receive a system.
Your company receives a way to keep creating systems with AI.
