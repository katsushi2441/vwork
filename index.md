# VWork バイブコーディングフレームワーク

VWorkは、経営者のPCに導入して使うバイブコーディング作業基盤です。

完成したシステムを外注して受け取るのではなく、経営者自身の課題をCodexと一緒に整理し、小さなコード、資料、Webページ、業務ツールへ変えていくためのフレームワークです。

## Blog Menu

### [VWork Blog](blog/)

VWorkを使ってバイブコーディングを始める方向けの情報です。
経営者向けの考え方、導入事例、業務改善、内製化、VWorkの使い方を蓄積します。

### [AI OSS技術解説](articles/)

Zenn連携を前提にした技術情報ブログです。
Codex、Claude、Ollama、AIエージェント、OSS、自動化、GitHub活用など、VWork利用者向けに限らない実装寄りの記事を蓄積します。
