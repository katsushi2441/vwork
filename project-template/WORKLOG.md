# WORKLOG

このファイルは、VWorkで行った作業、実行結果、エラー、次の依頼文を残すための共通テンプレートです。

VWorkでは、作ったものよりも「どう作り、どう動かし、次に何を改善するか」が重要です。

## WORKLOGに必ず残すもの

- 何を依頼したか
- どのファイルを変更したか
- どのコマンドを実行したか
- どこに出力されたか
- 結果を見てどう判断したか
- エラーがあれば何が起きたか
- 次にCodexへ何を頼むか
