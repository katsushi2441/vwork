# Use Case: Simple Web Tool

## Business Issue

A repeated internal calculation, lookup, or data conversion task is done by hand.

## First Tool

Create a local web page or small web app that:

- accepts input
- runs one business rule
- shows the result clearly
- can be improved later

## Input Examples

- pasted text
- uploaded CSV
- product URL
- order number
- customer code

## Output Examples

- `output/index.html`
- `src/app.py`
- `src/server.js`

See `prompts/web-tool.md`.
