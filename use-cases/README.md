# Use Cases

These use cases are starting points for business-owner vibe coding.

Each use case should become a small customer project:

```text
business issue
  -> sample data
  -> first tool
  -> output
  -> next improvement
```

## Recommended First Use Cases

- `sales-report.md`
- `excel-cleanup.md`
- `product-data.md`
- `sns-content.md`
- `inquiry-triage.md`
- `simple-web-tool.md`

Choose the use case closest to the customer's current pain, then copy the
relevant prompt from `prompts/`.
