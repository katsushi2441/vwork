# VWork Blog

VWork Blogは、バイブコーディングを企業内に導入するための考え方、実践知、営業・導入で得た気づきを蓄積する場所です。

ここに置く記事は、単なる日記ではなく、VWorkを改善し、顧客へ説明し、Codexが再利用できる知識資産として扱います。

## 方針

- 特定顧客の秘密情報、実データ、認証情報は置かない
- 顧客案件で得た知見は、匿名化・一般化して記録する
- 営業トーク、導入手順、設計判断、失敗から得た学びを残す
- 必要に応じて `docs/`、`prompts/`、`project-template/` に反映する
